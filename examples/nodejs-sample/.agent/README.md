# Agent System

This directory is the canonical instruction source for the Node.js sample repository.

Agents should read:

- `project-profile.md`
- `rulebase.md`
- `ownership.md`
- `gates.md`
- `roles/`
- `roles/prompts/`
- `workflows/`
- `runs/` when a non-trivial task needs a task-specific spec and plan
- `decisions.md`
- `lessons.md`

Tool-specific adapters must stay thin and point back here.
